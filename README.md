# Enigma

A C++ enigma machine simulator.

### Example use

Example use:
```
$ git clone https://github.com/finn-harman/Enigma.git
$ cd Enigma
$ ./enigma plugboards/I.pb reflectors/I.rf rotors/I.rot rotors/II.rot rotors/III.rot rotors/I.pos
```
